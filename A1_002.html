⚙️ 営業モード設定

顧客の注文方法・予約受付・AI注文支援チャンネルを設定します。これは最も重要なステップです。

💡 運用前に確認が必要な決定事項
  • WEBオーダー（LINE/Web）か APPオーダー（テーブルタブレット）、またはその両方？
  • 予約（Booking）を有効にするか？ → 詳細設定は Step 3 へ
  • ウェイトリスト（順番待ち）を有効にするか？ → 詳細設定は Step 3 へ
📲 注文受付チャンネルの有効化 Order Channel
📱 テーブルタブレット注文
テーブルに設置された画面（タブレット/キオスク）から顧客が直接注文できます。
🌐 LINE / Webオーダー
LINE Mini AppまたはWebサイトのQRコード読み取りから注文。顧客のスマートフォンで操作します。
⏰ ラストオーダー自動設定
営業終了前に自動でラストオーダー受付を終了します。指定時刻に注文チャンネルをロックします。
ラストオーダー締め切り時刻
営業終了の何分前に受付を締め切るか設定します
（営業終了 15分前）
📅 予約・ウェイトリスト受付 詳細設定は Step 3
📖 オンライン予約受付
LINE / Webから事前予約を受け付けます。予約の確認方法・支払い・キャンセルポリシーは Step 3 で設定します。
予約詳細設定へ (Step 3 - Tab 1)
🎫 ウェイトリスト（順番待ち受付）
満席時に順番待ちを受け付けます。LINE通知で呼び出し・自動キャンセルを設定します。詳細は Step 3 で設定します。
ウェイトリスト詳細設定へ (Step 3 - Tab 2)
🥡 テイクアウト注文受付
店外・事前注文のテイクアウトを受け付けます。受付時間・支払い方法は Step 3 で設定します。
🤖 AI注文受付設定（音声・文字） AI Order
🎤
Voice Order
音声注文支援
  • AIが音声で注文・メニュー案内を受け付ける
  • 多言語対応（日本語 / 英語 / 中国語）
  • テーブルタブレットのマイクを使用
  • 注文内容を音声で読み上げて確認
💬
Text Order
文字注文支援
  • AIがチャットテキストで注文を受け付ける
  • LINE / Web画面のチャットインターフェース
  • アレルギー・好みを考慮したおすすめ提案
  • 自然言語で「辛くないもの」など対応
AIプロフィール・会話シナリオ・おすすめエンジンの詳細設定 は Step 5 で行います
